# OmniRadiology

Welcome to the official open-source home of OmniRadiology. We build high-performance, privacy-first tools designed to streamline clinical radiology workflows and democratize access to advanced medical imaging technology.

## Core Projects

* **OmniRad Dashboard:** A modern, lightning-fast web interface built with Next.js for viewing DICOM images, managing studies, and generating diagnostic insights.
* **OmniRad Backend:** A high-performance Python ecosystem optimized for DICOM routing, server integration, and automated reporting workflows.

## Mission & Vision

Our goal is to bridge the gap between complex medical imaging infrastructure and modern web technologies. By keeping our core architecture completely open-source, we aim to provide healthcare developers and clinical researchers worldwide with secure, scalable, and customizable tools.
